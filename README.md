# Attendance Tracker

Программное решение для учета посещаемости лекционных занятий. Система позволяет вести базу данных студентов, групп и предметов, а также отслеживать пропуски в удобном графическом интерфейсе.

## Функциональные возможности

* Управление данными: добавление, редактирование и удаление записей о студентах, группах, предметах и датах.
* Учет посещаемости: ведение электронного журнала с фиксацией количества пропущенных часов.
* Визуализация: отображение данных в виде таблицы с цветовой индикацией пропусков.
* Экспорт: возможность выгрузки данных в формат JSON.
* Локализация: поддержка русского и английского языков интерфейса.

## Технологический стек

* Язык программирования: Python 3.13
* Графический интерфейс: Tkinter
* База данных: SQLite3
* Архитектура: MVC (Model-View-Controller)

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/ваш-репозиторий/AttendanceTracker.git
```

2. Запустите приложение:

```bash
python AttendanceTracking/src/main.py
```

## Структура проекта

* `src/models/` — логика работы с базой данных (SQLite).
* `src/views/` — графический интерфейс пользователя (окна, компоненты).
* `src/controllers/` — связующее звено между данными и интерфейсом.
* `src/utils/` — вспомогательные инструменты (валидаторы, локализация, сериализация).
* `data/` — файлы базы данных.
* `locales/` — файлы перевода (JSON).

Запуск через Docker
Приложение упаковано в Docker-контейнер. Для отображения интерфейса используется проброс X11-сервера хоста.

Подготовка (для Linux): Разрешите подключения к X-серверу:
```bash
xhost +local:docker
```
Запуск с помощью Docker Compose:

```bash
docker compose up --build
```

Примечание: для Windows и macOS требуется запущенный X-сервер (VcXsrv или XQuartz соответственно).

## Использование

1. При запуске программы откроется главное окно с таблицей посещаемости.
2. Используйте верхнее меню для настройки справочников (Группы, Студенты, Предметы).
3. С помощью фильтров выберите нужный год, месяц, группу и дисциплину.
4. Данные сохраняются автоматически при подтверждении изменений или через меню «Файл».
